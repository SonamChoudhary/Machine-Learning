TWO ARGUMENTS AT COMMAND LINE ARE REQUIRED:
a) Value of margin has to be given at command line. 
b) Filename of the files on which testing is desired.First training file then test file. 

RUNNING THE ALGORITHM:
a) Run the shell file. The margin for margin perceptron in the shell file is set to 1. To change, edit the shell file and set the desired margin value.
b) Command Line: Provide the required arguments as desired. 

NOTE:
Due to random initialization of the weights and bias, accuracies will differ.
